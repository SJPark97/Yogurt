1. DB 설계 유라
    1. 기능 명세서를 작성하라
    2. 개체(entity)와 속성을 추출
    3. 개체와의 관계는 여러 가지로 분류해서 정의
    4. 개념 설계 기반 ERD 생성
    5. 논리적 설계
        - 릴레이션 (Table로 변환)
    6. 물리적 스키마 구현
        - ERD 실제 테이블로 생성
    
    - 정규화
        - 무결성이 보장
        - 과다한 조인이 발생해 성능이 저하된다.
    - 반 정규화
        - SQL 작성 용이, 성능 향상 기대
        - 무결성이 깨질수 있다.


2. spring
    - 계층형 구조 사용
        -controller, web: 웹 계층
        -service: 비즈니스 로직, 트랜잭션 처리
        -repository: JPA를 직접 사용하는 계층, 엔티티 매니저 사용
        -domain: 엔티티가 모여 있는 계층, 모든 계층에서 사용
